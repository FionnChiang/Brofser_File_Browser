# 文件浏览器 (File Browser)

一个功能丰富的Android文件管理器应用，提供多标签浏览、文件操作、媒体预览和自定义设置。

## 功能特性

### 📁 文件浏览
- **多标签管理**：支持同时打开多个文件夹标签页
- **多种视图模式**：列表视图、网格视图、画廊视图、无图标列表视图
- **面包屑导航**：清晰的路径导航，支持快速跳转
- **左右滑动切换标签**：手势操作便捷切换标签页
- **随机文件夹**：随机进入当前目录的子文件夹

### 🔍 搜索功能
- **实时搜索**：输入时实时过滤文件
- **高级过滤**：按文件类型（全部、文件夹、图片、视频、音频、文本、其他）、文件大小、修改日期筛选
- **全局搜索**：支持在整个存储中搜索
- **搜索结果导航**：点击搜索结果可直接跳转到文件所在目录

### ✂️ 文件操作
- **多选模式**：支持批量选择文件进行操作
- **基本操作**：复制、剪切、粘贴、删除、重命名
- **高级操作**：压缩为ZIP、隐藏文件、创建快捷方式、文件属性查看
- **分享功能**：支持分享单个或多个文件
- **文件属性**：显示文件详细信息（大小、类型、修改时间、权限等）

### 🎨 媒体预览
- **图片查看器**：支持缩放、滑动浏览
- **视频播放器**：全屏播放视频文件
- **音频播放器**：播放音频文件
- **文本查看器**：查看文本文件内容

### ⚙️ 自定义设置
- **背景自定义**：支持设置自定义背景图片和透明度
- **视图偏好**：可设置默认视图模式和排序方式
- **显示设置**：是否显示隐藏文件、文件夹优先排序
- **书签快捷方式**：可添加常用目录到侧边栏快捷方式

### 🗂️ 其他功能
- **侧边栏导航**：快速访问存储、下载、图片、音乐、视频等系统目录
- **权限管理**：兼容Android 11+的存储权限管理
- **文件提供器**：支持通过FileProvider分享文件
- **多语言支持**：中文界面

## 技术架构

### 开发环境
- **编程语言**：Java
- **最低Android版本**：Android 5.0 (API 21)
- **目标Android版本**：Android 14 (API 34)
- **构建工具**：Gradle

### 主要依赖
- `androidx.appcompat:appcompat:1.6.1`

## 安装与运行

### 前提条件
1. Android Studio 2023.1.1 或更高版本
2. Android SDK 34
3. Java 8 或更高版本

### 构建步骤
1. 克隆项目到本地
2. 使用Android Studio打开项目
3. 等待Gradle同步完成
4. 连接Android设备或启动模拟器
5. 点击运行按钮（▶️）或按 `Shift+F10`

### 权限说明
应用需要以下权限：
- **Android 12及以下**：`READ_EXTERNAL_STORAGE`、`WRITE_EXTERNAL_STORAGE`
- **Android 13+**：`READ_MEDIA_IMAGES`、`READ_MEDIA_VIDEO`、`READ_MEDIA_AUDIO`
- **Android 11+**：`MANAGE_EXTERNAL_STORAGE`（完整文件访问权限）

首次运行时会请求必要的文件访问权限。

## 使用指南

### 基本操作
1. **浏览文件**：点击文件夹进入，点击".."返回上级目录
2. **打开文件**：点击文件使用对应应用打开
3. **多选操作**：长按文件进入多选模式，或通过菜单进入
4. **新建标签**：点击标签栏的"+"按钮
5. **关闭标签**：点击标签上的"×"按钮

### 搜索功能
1. 点击工具栏搜索图标进入搜索模式
2. 输入搜索关键词实时过滤
3. 点击筛选按钮设置高级搜索条件
4. 点击搜索结果跳转到文件所在目录

### 文件操作
1. **复制/剪切**：多选文件后点击相应按钮
2. **粘贴**：导航到目标目录后点击粘贴按钮
3. **压缩**：多选文件后点击"更多"→"压缩为ZIP"
4. **分享**：多选文件后点击分享按钮
5. **属性查看**：多选单个文件后点击"更多"→"属性"

### 视图设置
1. **切换视图**：点击菜单→"更多"→"视图切换"
2. **排序方式**：点击菜单→"更多"→"排序方式"
3. **显示设置**：进入设置界面调整

## 开发说明

### 代码规范
- 使用Java标准命名规范
- 类名使用大驼峰式命名
- 变量和方法名使用小驼峰式命名
